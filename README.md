# Typological Investigation of Universal 1084

Данный репозиторий посвящен исследованию лингвистической универсалии **Universal 1084**, касающейся структуры инвентаря смычных согласных в языках мира.

## Предмет исследования: Universal 1084
Универсалия утверждает: 
> *«Если в языке отсутствует полная шестипозиционная парадигма смычных (/p, t, k, b, d, g/), то лакуны (пропуски) вероятнее всего возникнут в позиции /g/, затем /p/, далее /d/ и, наконец, /b/.»*

## Основные этапы работы

### 1. Балансировка выборки
Исходная база содержит около 600 языков, что может приводить к генетическим или ареальным перекосам (например, доминирование австронезийских языков). 
В проекте реализован алгоритм **стратифицированного сэмплирования**:
* За эталон взято распределение языковых семей по макроареалам из **Glottolog**.
* Для каждой пары «семья — ареал» рассчитаны квоты.
* Итоговая выборка ($N \approx 100$) формируется путем случайного отбора из исходных данных согласно этим квотам.

### 2. Фильтрация данных
Для корректной проверки универсалии из анализа исключены:
* Языки с полной шестеркой взрывных (в них нет лакун для исследования).
* Языки без оппозиции по звонкости (только глухие или только звонкие взрывные), так как в них отсутствие звука обусловлено отсутствием всего ряда, а не системной лакуной в парадигме.

### 3. Статистический анализ
Анализ частотности пропусков в позициях `/p, t, k, b, d, g/` для подтверждения или опровержения иерархии вероятностей, предложенной в Universal 1084.

## Структура репозитория

* `our_data.csv` — исходный массив данных.
* `family_area_pairs_counts.csv` — статистика распределения языков в мире (Glottolog).
* `family_area_pairs_counts_my.csv` — статистика распределения в исходной выборке.
* `final_balanced.csv` — итоговая сбалансированная и отфильтрованная выборка.
* `typology.ipynb` — Jupyter Notebook с кодом балансировки, фильтрации и визуализации результатов.
* `languages.csv`, `typology.tsv` — вспомогательные таблицы с данными по языкам и их признакам.

## Как запустить
1. Клонируйте репозиторий.
2. Установите зависимости: `pip install pandas numpy matplotlib`.
3. Откройте `typology.ipynb` для воспроизведения расчетов.
